A complete Trip Management solution for WordPress. Create, organize, and display trips with powerful filters for destinations, price, and duration.

== Description ==

TripOrganizer is a powerful WordPress plugin designed for travel agencies, tour operators, and bloggers who want to showcase and manage trips easily.

**Features:**
* Custom Post Type for Trips
* Destination taxonomy for trip locations
* Custom meta fields (Price and Duration)
* Advanced filter with AJAX for:
  - Destination
  - Price range
  - Duration range
* Shortcode `[trip_organizer_filter]` for displaying filters and trips
* Fully responsive and lightweight design
* Built for speed and SEO-friendly

Perfect for creating a **travel booking directory** or **tour listing platform**.

== Installation ==

1. Download the plugin ZIP file and upload it to your WordPress site.
2. Go to **Plugins > Add New** and click **Upload Plugin**.
3. Activate **TripOrganizer** from the Plugins menu.
4. Navigate to **Trips > Add New** to create your first trip.
5. Use the shortcode `[trip_organizer_filter]` on any page to display the filter and trip listing.

== Frequently Asked Questions ==

= How do I display the trip filter on my page? =
Use the shortcode `[trip_organizer_filter]` on any post or page.

= Can I add more destinations? =
Yes, you can add destinations under **Trips > Destinations** in the WordPress admin.

= Does it support custom themes? =
Yes, the plugin works with any theme. You can override the CSS for a custom look.
